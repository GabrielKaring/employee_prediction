# Employee Prediction
Introduz um fluxo de trabalho completo de ciência de dados para a previsão de rotatividade de funcionários, incluindo a formulação do problema, EDA (Análise Exploratória de Dados), seleção de variáveis, pré-processamento de dados, treinamento de modelos (Regressão Logística, Árvore de Decisão, Random Forest), avaliação e interpretação. Adiciona documentação detalhada em markdown, além de código para carregamento, limpeza e visualização dos dados, comparação de modelos e análise da importância das variáveis.
